🚀 Gestion de Questionnaire QCM en Ligne
Application web complète de gestion d’examens QCM développée avec JSP, PHP et TailwindCSS
📌 Aperçu du projet
Cette application permet de digitaliser entièrement le processus d’examens à travers une plateforme interactive de Questionnaires à Choix Multiples (QCM).
Elle simule un environnement réel d’évaluation avec un portail étudiant et un dashboard administrateur avancé.
🎯 Objectif
Concevoir une application web complète permettant :
La gestion des examens en ligne
L’évaluation automatique des étudiants
Le suivi des performances en temps réel
L’administration centralisée des sessions et des questions
👨‍🎓 Portail Étudiant
🔐 Authentification via numéro étudiant unique
⏳ Salle d’attente dynamique avec statut de session en temps réel
📝 Passage d’examen QCM chronométré
🎲 Questions tirées aléatoirement depuis la base de données
📊 Historique des performances personnelles
📧 Envoi automatique des résultats par e-mail (SMTP)
👩‍💼 Espace Administrateur
📊 Dashboard analytique (statistiques globales)
👨‍🎓 Gestion des étudiants (CRUD complet)
📅 Planification des sessions d’examens
❓ Gestion de la banque de questions QCM
🏆 Classement général des étudiants (Leaderboard)
🔍 Recherche et filtres avancés
⚙️ Fonctionnalités clés
✔ Système d’examen en ligne complet
✔ Génération dynamique des questions
✔ Correction automatique des QCM
✔ Envoi de résultats par email
✔ Classement des étudiants par performance
✔ Interface responsive et moderne
🛠️ Technologies utilisées
JSP (Java Server Pages) – logique serveur
PHP – traitement backend
MySQL – base de données
TailwindCSS – design moderne et responsive
SMTP – envoi d’emails automatiques
